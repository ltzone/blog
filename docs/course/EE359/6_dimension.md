## Singular Value Decomposition


$$
A_{m\times n} = U_{m \times r} \Sigma_{r\times r}  (V_{n \times r})^T
$$

- $A$: Input Data matrix
- $U$: Left Singular Vectors
- $\Sigma$: Singular Values, diagonal
- $V$: Right Singular Vectors
